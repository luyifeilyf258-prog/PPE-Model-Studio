# Changelog

## [1.0.0] - 2026-07-03

### Added

- Initial release of PPE Model Studio.
- Added YAML-based parameter configuration.
- Added two-dimensional particle/binder electrode structure generation.
- Added interactive structure preview.
- Added COMSOL Java model export.
- Added COMSOL compile and batch run workflow.
- Added COMSOL path configuration and validation in the GUI.
- Added run-based output folder structure.
- Added CSV result import and voltage curve plotting.
- Added PyInstaller folder-based distribution workflow.
- Added Inno Setup installer workflow.

### Notes

- COMSOL is required only for Java compilation, MPH model generation, and batch simulation.
- Basic parameter editing, structure generation, preview visualization, and CSV plotting can be used without COMSOL.
